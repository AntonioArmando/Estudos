formatações de códigos

código

usamos a tag <code> para pre formatarmos um código em html.
exemplo:

System.out.println("Alo pessoal!");

A tag <code> cria espaço entre as letras que compõem o código facilitando o leitura da mesmo.

pre formatações

a tag <pre> permite permite colocarmos um código pre formatado.
exemplo: 

                i = 4

                while i < 8:
                    print(i)
                    i += 1
